NewStore Associate App allows store associates to sell products in-store and on the endless aisle, using customer profiles and purchase history to leverage better sales and customer relationships.
Features
Documentation
Configuration
Set up NewStore Associate App to manage the printers that you use in your store during the checkout process.
Prerequisites
Configuring printers
Configuring printers using the legacy framework
Migrating printers from the legacy framework
Customization
Modify NewStore Associate App to match your brand’s look and feel and include new features.
Customizing NewStore Associate App
Distribution
As per Developer Enterprise Program usage guidelines by Apple, all NewStore apps are being distributed using Apple Business Manager.
Distributing Associate App
Login
If your business has enabled logging into NewStore Associate App with your existing corporate credentials, use them to log into the app. Otherwise, use your assigned NewStore credentials.
Using NewStore Associate App
Customers
You can use NewStore Associate App for clienteling purposes and also for managing customer profiles.
Using NewStore Associate App
Clienteling
Associates can use NewStore Associate App to offer clienteling support to customers, if the feature has been set up for their store.
Using NewStore Associate App for clienteling
Cart
You can use NewStore Associate App for creating, saving, and restoring shopping carts.
Using NewStore Associate App
Selling
Cash Management
You can use NewStore Associate App to manage cash in the store, including daily open, cash sale and returns, audit count and daily close.
Managing cash in the store
Payment Service Providers
NewStore allows you to connect payment devices or terminals to NewStore Associate App and set up a payment service provider (PSP) for use in your stores to manage payments and refunds.
Setting up payment service providers
Offline Mode
NewStore allows you to use limited features of NewStore Associate App when the device is not connected to the internet or there is no internet connectivity in the store.
Using NewStore Associate App when offline
Fulfillment
Associates can use NewStore Associate App to fulfill orders, if the feature has been configured for their store.
Inventory
Associates can use NewStore Associate App to manage inventory, if the feature has been configured for their store.
Using NewStore Associate App to manage inventory
