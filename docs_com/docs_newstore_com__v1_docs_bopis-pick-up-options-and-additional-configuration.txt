To successfully enable a store for Buy Online Pick Up In-Store (BOPIS), proper setup is crucial. The platform utilizes a Geographic Coordinate System to pinpoint the nearest store for a given request, making it essential to configure the store with accurate Latitude and Longitude attributes. This ensures that customers are directed to the correct store location, providing a seamless BOPIS experience.
Refer
Module 1 - Foundational Setup for store setup
The
In-store pickup
config allows you to toggle Buy Online Pick Up In-Store (BOPIS) support for a store on or off, providing greater flexibility in managing your store's capabilities. This means you can easily enable or disable BOPIS as needed, giving you more control over your store's operations and allowing you to go live or make changes quickly and effortlessly.
Important
The
In-store Pickup API
can configure the global configurations like the default_search_radius, display_name, and default_limit. This API enables store-level configuration, allowing you to customize store settings and preferences.
