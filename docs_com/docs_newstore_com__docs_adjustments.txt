Types of inventory adjustments
There are three types of inventory adjustments:
Between virtual locations
From a virtual location to an untracked location
From an untracked location to a virtual location
Each adjustment type affects inventory tracking and availability. Adjustments involving sellable and unsellable virtual locations directly impact Available-to-Promise (ATP) quantities.
Moving inventory from a sellable to an unsellable location reduces ATP
Moving inventory from an unsellable to a sellable location increases ATP
