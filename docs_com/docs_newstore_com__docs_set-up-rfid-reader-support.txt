Enable RFID reader support in the NewStore Associate App to identify products and capture the Electronic Product Code (EPC) during checkout.
Important
Available as
Early Access
with Associate App 1.66.0 and only with
Checkpoint CPiD 400 POS Reader
. Contact your Omnichannel Success Manager for more information.
To enable the new RFID reader set up in Associate App, contact your Omnichannel Success Manager.
Go to the
Associate App configuration API
.
Set
release_toggles
>
new_printer_management
to
true
.
Setting up an RFID reader for checkout
Follow these steps on each iOS device that requires RFID scanning to identify products and capture the EPC:
Tap the navigation drawer menu >
Settings
>
Checkpoint RFID Scanner
.
Enable the
Checkpoint RFID Scanning
toggle.
Type the IP address of the RFID reader.
Select the antenna that will be connected to the iOS device.
Tap
Test Connection
button.
On success: a green ‘Connection Successful’ message appears at the top of the screen.
On failure: a red ‘Connection Failed’ message appears. Verify the IP address and antennas, and try again.
Go to the cart view to see the ‘ X Stop RFID Scanning’ banner. This shows the connection is established and the reader is ready to pass the product information to the app.
