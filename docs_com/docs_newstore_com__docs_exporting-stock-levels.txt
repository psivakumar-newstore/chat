You can export stock levels as a CSV file. The export contains all the data that is displayed in the
Stock Levels
tab. For more information, see
Monitoring stock levels
.
To export the stock levels:
Go to the
Catalog
>
Stock Levels
tab.
Set the filters to display the products that you want to export.
Click
Export
.
