Important
Managing fiscal stores requires the stores to be already created in the NewStore platform. For more information on setting up a store see
Tutorial: Setting up a store
.
Permission required
This action can be performed by users with the Manage fiscal stores permission.
NewStore enables you to operate stores in countries where fiscal regulations apply. By using Omnichannel Manager, you can:
View stores in supported fiscal countries.
Fiscally activate cashboxes in countries where software fiscal regulations apply.
View fiscal cashboxes in countries where software fiscal regulations apply.
View fiscal printers in countries where hardware fiscal regulations apply.
For more details on supported fiscal countries, see
About internationalization
.
Viewing fiscal stores
To view fiscal stores set up for your business in supported fiscal countries, go to
Settings
>
Fiscalization
>
Fiscal stores
.
The following columns are available:
Store ID
: Identifier of the store.
Store Label
: Name of the store.
Country
: Country where the store is located.
Status
: Fiscal status of the store. It indicates whether the store has fiscal cashboxes/printers that allow the store to fiscalize in-store transactions.
Viewing fiscal features for a store
To view a fiscal cashbox or a fiscal printer in a store:
Go to
Settings
>
Fiscalization
>
Fiscal stores
.
Click the store ID of the fiscal store for which you want to view fiscal details.
The
Fiscal Stores
overview page displays the following information:
Overview
:
Store ID
: Identifier of the store.
Country
: Country where the fiscal store is located.
Timezone
: Timezone of the store. Timezone will be used for determining the time of periodic closures.
Status
: Fiscal status of the store. It indicates whether the store has fiscal cashboxes or printers that allows the store to fiscalize in-store transactions.
For stores where software fiscal regulations apply, the status field can display either
Active
or
No active cashbox
.
For stores where hardware fiscal regulations apply, the status field can display either
{#} active fiscal printer(s)
or
No active fiscal printer
.
Cashboxes
, available for stores in countries where software fiscal regulations apply:
Cashbox ID
: Cashbox identifier. It is generated automatically during the fiscal activation process. Click the cashbox ID to view fiscal activation information for the cashbox in the
Cashbox Overview
screen.
Fiscal Provider
: Fiscal provider that is used for fiscalizing transactions.
State
: Indicates if the cashbox is active or deactivated.
Activated At
: Date and time when the cashbox was fiscally activated.
Deactivated At
: Date and time when the cashbox was deactivated.
Fiscal Printers
, available for stores in countries where hardware fiscal regulations apply:
Serial Number
: Serial number of the fiscal printer.
Name
: Name of the fiscal printer, assigned by the user when configuring the fiscal printer.
Manufacturer
: Manufacturer of the fiscal printer.
Model
: Model of the fiscal printer.
IP Address
: IP address of the fiscal printer.
State
: Indicates if the fiscal printer is active or deactivated.
Configured At
: Date and time when the fiscal printer was configured. This timestamp is updated when the fiscal printer is configured again.
Activating fiscal cashboxes
You can activate fiscal cashboxes through Omnichannel Manager for stores operating in countries where software fiscal regulations apply.
Note
In countries where hardware fiscal regulations apply, fiscal printers can be configured through Associate App, and consequent changes can be viewed in Omnichannel Manager. See
(Optional) Setting up a fiscal printer
.
To activate a fiscal cashbox:
Go to
Settings
>
Fiscalization
>
Fiscal stores
.
Click on the store ID of the fiscal store.
Click
Activate Cashbox
to open the fiscal activation drawer.
Important
In countries where software fiscal regulations apply, a store can have only 1 active fiscal cashbox. To activate a new cashbox, the existing active cashbox must be deactivated. Talk to your point of contact at NewStore to deactivate the active cashbox.
Enter fiscal activation information.
Important
Fiscal activation information cannot be changed after the cashbox is fiscally activated.
Click
Activate Cashbox
.
In the confirmation dialog box, ensure that the specified fiscal activation information is correct. After validating the information, click
Confirm
.
After the fiscal activation is completed, the fiscal cashbox is displayed in the
Fiscal Stores
overview page.
