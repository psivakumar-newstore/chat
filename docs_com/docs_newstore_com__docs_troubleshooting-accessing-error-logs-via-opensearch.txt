When setting up integrations or configuring features in the NewStore platform, exceptions may occur that differ from the expected setup behavior.
When the generated errors are not explicit via error codes or API response payloads, partners can access existing logs via the NewStore OpenSearch instance.
Important
Access to this tool is controlled via Okta. If you do not have SSO credentials, create a ticket in the
Implementation Helpdesk
or
send us an email
.
