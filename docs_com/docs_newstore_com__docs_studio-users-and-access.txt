There are two role types in Studio: User and Admin. Each role is assigned per app. If your company manages multiple apps, the same user can have the same role across different apps.
User
Users can access and edit any content in Studio
Users cannot access Manage Users page
Cannot invite, delete, view users assigned to the app
Admin
Admins can access and edit any content in Studio.
Admins can manage users - invite, delete, view users assigned to the app
Multiple admins can exist for the same app.
Note:
To promote a user to Admin, contact your Omnichannel Success Manager or Support for security reasons.
Manage users
Note:
Only users with an Admin role can manage other users.
To manage users:
Log in with an Admin account.
Click the
Account
icon in the top-right corner.
Select
Manage Users
from the dropdown.
From the
Manage Users
page, you can:
View all users assigned to the app
Invite new users via email
Delete existing users
