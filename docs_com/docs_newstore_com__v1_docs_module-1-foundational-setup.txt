Dodici, a renowned luxury fashion brand based in Milan, Italy, has partnered with us to ensure a seamless migration to the
NewStore platform. As part of this transition, the
VP of Retail
has outlined key requirements aimed at elevating the customer experience:
Support for
cash and carry
transactions
Enablement of
cash
as a payment method
To fulfill these needs, we’ll configure
returns and exchanges
within the NewStore platform, delivering a seamless, customer-centric experience that upholds
Dodici’s high standards of luxury and service.
This tutorial covers the initial setup and walkthrough the basic configuration that is required in all projects, regardless of the agreed scope. At the end of this tutorial, you should be able to perform a simple cash and carry transaction.
When implementing NewStore the following steps will always be required and should follow this process flow to keep the setup as efficient as possible. This involves
Loading Products into the NewStore Platform. In this section, we'll cover typical import patterns.
Loading Prices for those items. In this section we will cover steps around reconciling the products and prices loaded.
Creating Store. This section will cover basic data needed to define a store.
Creating Availability. In this section we will assume that product availability is mastered by an external system. However, in this section we cover how this is important in the creation of fulfillment locations.
Configuring Tax. In this section we'll talk the different types of configuration but will only be able teach the simplest form of tax configuration.
Configure Application. This covers the required configuration needed to get the application to work before you can perform a single transaction.
