API to create, read, update and delete notes for orders or items in NewStore Omnichannel Cloud.
Get notes by order ID
Returns all notes associated with the specified order ID in the request URL sorted from newest to oldest.
This method also returns all the notes for the items that are part of the specified order.
Authorizations:
newStoreAccessToken
path
Parameters
order_uuid
required
string
Unique ID for an order.
Responses
200
Successfully processed.
400
There is an error in the url.
404
The endpoint could not be found.
500
An error occurred on the server.
default
Unexpected Error

Create order note
Creates a new note that applies to the order ID specified in the request URL. For example,
create a note for an order that has been cancelled by the customer in the store.
Authorizations:
newStoreAccessToken
path
Parameters
order_uuid
required
string
Unique ID for an order.
Request Body schema:
application/json
text
required
string
[ 1 .. 4000 ] characters
The content of this note.
source
required
string
[ 1 .. 256 ] characters
The identifier for the author of the note. This field only supports
user_id
.
source_type
string
[ 1 .. 256 ] characters
The type of the source, e.g. 'user', 'system'
tags
required
Array of
strings
[ items
[ 1 .. 128 ] characters
]
The tags associated with this note.
Responses
200
Note has been successfully created.
400
The request body is invalid.
404
The endpoint could not be found.
500
An error occurred on the server.
default
Unexpected Error
