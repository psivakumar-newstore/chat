profiles-management
Merge two profiles into one.
Merges data from specific fields from source profile into target profile.
Mergeable fields are -
first_name
,
last_name
,
phone_number
,
extended_attributes
,
addresses
.
After merging data, source profile will be deleted as a result of the operation.
Authorizations:
bearerAuth
Request Body schema:
application/json
source_profile_id
required
string
ID of the source profile.
target_profile_id
required
string
ID of the target profile.
Responses
200
Profiles were successfully merged.
400
Bad Request -
source_profile_id
or
target_profile_id
are empty or missing.
404
Not found - cannot find profiles identified by
source_profile_id
or
target_profile_id
.
500
A generic error response.
