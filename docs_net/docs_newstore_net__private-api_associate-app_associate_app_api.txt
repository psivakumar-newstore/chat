query
Parameters

string
A language-region pair in form of
<2 letter language code name from ISO 639-1>-<2 letter country subdivision code from ISO 3166-2>
identifier used to differentiate translated content in API responses.
Responses
200
Returns the data and permissions of the associate itself, its associated store and the tenants returns configuration.
400
Bad request - Request data is invalid.
Use the error codes for error handling. The error messages should never
be used for error handling as they can change at any time.

missing_store_assignment
call_validation_error
404
No associate or store information was found.

employee_not_found
store_not_found
