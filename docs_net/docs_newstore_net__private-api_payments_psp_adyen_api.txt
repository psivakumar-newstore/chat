(
1.0
)
Download OpenAPI specification

Download
This API will be used by Adyen to forward payment events. The API represents Adyen configuration
for stores in your business, including notifications reflecting a payment status update.
The
Register payment status update
method provided under this resource must be
added as webhooks to your Adyen setup
.
Register payment status update
This method must be
called by Adyen
.
This interface conforms to Adyen's notification structure.
Request Body schema:
application/json
live
required
string
Specifies whether the event that triggered the notification happened on the test or live environment.
notificationItems
required
Array of
objects
List of payment notifications. The list contains a single item.
Responses
200
The notification was processed.
401
Authorization header didn't match the expectation.
403
Could not validate message signature.
