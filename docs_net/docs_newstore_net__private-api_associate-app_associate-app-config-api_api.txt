Modify Associate App configuration
Changes the Associate App configuration. Replaces complete configuration. If configuration does not  exist for a tenant, it will be created.
Responses
200
Returns the new value of the configuration.
400
Invalid JSON. Received configuration could not properly parsed.
422
Invalid schema. Received configuration is not valid against the configuration schema.
