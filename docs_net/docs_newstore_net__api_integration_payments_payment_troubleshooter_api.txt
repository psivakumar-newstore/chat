Troubleshoot NewStore Checkout configuration
Returns a summary of the NewStore Checkout configuration checks.
Authorizations:
newStoreAccessToken
Request Body schema:
application/json
cart_token
string
Base64 cart token, use one from a NewStore Checkout URL.
Responses
200
NewStore Checkout troubleshoot summary.
