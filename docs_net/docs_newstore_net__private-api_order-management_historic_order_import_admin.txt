Use this API to import historic order into NewStore and create a sales_order object out of it. This is an internal
API and should not be used outside of OM.
Use this API to import historical order into NewStore and create a sales_order object.
Import historic orders into NewStore and creates a sales_order object out of it. Other services are notified
of this imported order by emitting a event.
