Financial communication configuration API
(
1.0.0
)
Download OpenAPI specification
:
Download
Use this API to configure notifications and financial communications that are sent from the
NewStore platform to customers in your business. This may include receipts and other similar
financial documents.
Note
: Financial documents and notifications can be sent to the customer via email or
through push notifications. Emails contain the relevant financial documents as attachments.
Financial communication configuration
Use this resource to manage the financial communication configuration for your business.
Get financial communication configuration
Retrieves the complete financial communication configuration for your business.
The response payload contains the current version of the configuration, the timestamp when
it was last updated, and who updated it.
Authorizations:
newStoreAccessToken
header

ID of the request. Generated automatically.

required
string

Current configuration
404
Tenant or service not found
500

Updates the current financial communication configuration for your business.
Authorizations:
newStoreAccessToken
query
Parameters

required
string
Current version of the financial communication configuration.
Use the
Get financial communication configuration

header
Parameters

ID of the request. Generated automatically.
tenant
required
string

string
ID of the user. Substituted automatically from the auth token.
Request Body schema:
application/json
The updated config value
notification_options
required
object
Notification options related to financial documents.
push_notification_channel_type
required
string
Enum
:
"customer_uuid"
"email"
Push notification need a channel (type of addressing) for where to send notifications. That can be the internal customer identifier or their email.
options
required
object
A map of enabled notifications and their respective channels.
Responses
200
The updated configuration
400
Invalid request
404
Tenant or service not found
409
The version key is out of date, configuration was updated
500
An internal error occurred
