Use this API to store the Adyen integration credentials. For more information on integrating
Adyen with NewStore, see
this guide
.
Update credentials for Adyen payment devices and APIs
Updates the set of credentials used to authenticate Adyen payment devices and APIs.
Authorizations:
newStoreAccessToken
Request Body schema:
application/json
company
required
string
non-empty
The company part of the User Name configured in Adyen. If the Adyen User Name is
ws_12345@Company.Example
then the company is
Company.Example
.
username
required
string
non-empty
The user part of the User Name configured in Adyen. If the Adyen User Name is
ws_12345@Company.Example
then the username is
ws_12345
.
password
required
string
non-empty
The password configured in Adyen.
account
required
string
non-empty
The Adyen merchant account to be used as the default.
Responses
200
The credentials were successfully updated.
