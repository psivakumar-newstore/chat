Retrieves all orders for a customer email address.
Use the
GET: /consumer_profiles
method with
?q=jay.doe@example.com
, to
retrieve the customer's
id
. Note that based on your query, multiple results may be returned.

Retrieve orders associated with the order ID
Retrieves the order associated with the provided order ID. The ID can be internal to NewStore or external.
Authorizations:
newStoreAccessToken
path
Parameters
type
required
string
Default:
"external_id="
The type of the identifier for the order.
Use
id=
to use the NewStore Omnichannel Cloud ID, or
external_id=
to use an external ID.
id
required
string
The identifier for the order either inside NewStore Omnichannel Cloud (id) or in an external system such as
Shopify.
Responses
200
The order with the provided ID is returned.
400
Missing or malformed input parameters.
404
No order was found for the internal or external order ID or the provided zip/country/email
does not match the order associated with the external order ID.
