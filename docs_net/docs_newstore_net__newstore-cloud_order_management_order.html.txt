This API allows you to retrieve order level information. Currently, the API allows you to
retrieve a list of up to 2000 fulfillment requests for an order.
This includes fulfillment requests from a store or DC.
You can also use this resource to check the status of each item in the fulfillment request, and
perform a reconciliation with your ERP.
To accept, reject, or retrieve the status of fulfillment requests, see the
Fulfillment provider API
.
List fulfillment requests for an order
Returns a list of up to 2000 fulfillment requests for an order. Use this method to check
the status of items in a fulfillment request for an order ID.
